Coloca aquí tus iconos personalizados para la aplicación.

Convenciones de nombres sugeridas:

1) Icono de Ajustes (Engranaje)
   - settings.svg   (recomendado) o settings.png
   - Opcional para modo oscuro: settings_dark.svg / settings_dark.png

2) Icono de Volver (Chevron/Back)
   - back.svg   o back.png
   - Opcional para modo oscuro: back_dark.svg / back_dark.png

Detalles:
 - Se detectan automáticamente estos archivos.
 - Si existe la variante "_dark" y el tema está en oscuro, se usa esa.
 - Si no se encuentra ningún archivo, la app genera iconos vectoriales por defecto.
 - Tamaño recomendado del lienzo: 24–32 px con fondo transparente.

No es necesario reiniciar la app: cambia de tema (Ctrl+D) para forzar refresco o reabre la ventana.
